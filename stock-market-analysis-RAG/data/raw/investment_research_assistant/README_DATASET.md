# Investment Research Assistant Dataset (Lab 2)

Tickers included: NVDA, AMD, TSLA

Per ticker resources collected:
- Latest 10-K (SEC)
- Latest 10-Q (SEC)
- Latest earnings call transcript (Motley Fool transcript page)
- Latest investor material/presentation proxy (SEC 8-K company investor materials)

Important:
- Transcript pages are saved as HTML source and may contain additional site chrome.
- Investor materials use official SEC-filed 8-K exhibits to ensure stable access.
- See `dataset_index.csv` for exact file paths and source URLs.
- `comparables.csv` contains basic comparables metrics derived from SEC companyfacts.
